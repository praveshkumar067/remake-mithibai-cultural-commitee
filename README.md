# 🎭 Mithibai Cultural Committee (MCC) — Landing Page

A premium, modern, and visually stunning landing page for **SVKM's Mithibai Cultural Committee (MCC)**. Built with rich animations, glassmorphic dark-theme aesthetics, and a fully responsive layout — this portal showcases the committee's 9 departments, the flagship Kshitij Festival, achievements, and features a live audition registration system.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| **9 Department Hub** | Interactive card grid with detail-injection modals for Dance, Drama, Music, Fashion, Fine Arts, Literary Arts, Business Events, Gaming & Sports, and Graphics & Photo Media. |
| **Live Countdown Timer** | Real-time countdown to the annual *Kshitij '26* festival with animated timer blocks. |
| **Audition Registration Form** | Validated form with custom-styled inputs, department selector, and animated success overlay. |
| **Glassmorphic Dark Theme** | Neon violet, cyan, and gold accents with backdrop-blur glass panels and smooth gradient text. |
| **Mobile Responsive** | Collapsible hamburger navigation, adaptive grids, and touch-friendly interactions. |
| **Scroll Reveal Animations** | Elements animate into view as users scroll through sections. |
| **Disclaimer Compliant** | Clear student-run disclaimers regarding college affiliation and data privacy. |

---

## 🛠️ Tech Stack

- **HTML5** — Semantic, accessible markup with SEO meta tags
- **CSS3** — Vanilla CSS with custom properties, flexbox/grid layouts, keyframe animations, and cross-browser compatible `background-clip` 
- **JavaScript (ES6+)** — Vanilla JS for countdown timer, modal logic, form validation, scroll reveals, and mobile menu

No frameworks. No build tools. Pure front-end.

## 🚀 Getting Started

This is a static site — no install or build step required.

### Option 1: Python
```bash
cd antigravity-superstation
python -m http.server 8000
```
Then open → **http://localhost:8000**

### Option 2: Node.js
```bash
npx serve .
```

### Option 3: Just open the file
Double-click `index.html` in your file explorer to open it directly in your browser.
